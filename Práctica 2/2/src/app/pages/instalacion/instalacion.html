Instalación del entorno Angular

En esta sección se explican los pasos necesarios para preparar el entorno de desarrollo de Angular. Instalaremos Node.js, npm y la herramienta oficial Angular CLI, todo lo necesario para poder crear y ejecutar proyectos con Angular.

1. Instalación de Node.js

Angular utiliza Node.js para ejecutar scripts y gestionar dependencias. Node incluye además npm, el gestor de paquetes que permite instalar Angular CLI.

  1. Accede a la página oficial: https://nodejs.org
  2. Descarga la versión LTS.
  3. Ejecuta el instalador y sigue el asistente (Siguiente → Siguiente → Instalar).
  4. Tras la instalación, abre una terminal (CMD, PowerShell o VS Code) y verifica la instalación:

node -v

npm -v


Si ambos comandos devuelven un número de versión, la instalación se ha realizado correctamente.

2. Instalación de Angular CLI

Angular CLI es la herramienta oficial para crear, configurar y ejecutar proyectos Angular. Es necesaria para trabajar con Angular de forma cómoda y ordenada.

Para instalarla, ejecuta este comando en la terminal:

npm install -g @angular/cli


La opción -g indica que la instalación es global, es decir, estará disponible en todo el sistema.

Para comprobar que Angular CLI se instaló correctamente ejecuta el siguiente comando:

ng version


3. ¿Qué se ha instalado?

  • Node.js — entorno necesario para ejecutar Angular.
  • npm — gestor de paquetes que permite instalar librerías.
  • Angular CLI — herramienta para crear y gestionar proyectos Angular.

Con todo esto instalado, el entorno ya está listo para crear el primer proyecto de Angular, como se verá en la siguiente sección del tutorial, puedes ir al apartado Tutorial en el menú de arriba.
